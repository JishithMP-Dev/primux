# Primux

**Primux** is a comprehensive, lightweight, and well-structured JavaScript utility library that extends JavaScript’s standard capabilities.  
It provides a wide range of reusable helper functions for **functions, arrays, objects, strings, numbers, DOM, browser APIs, and more** — all in one clean package.

Primux is designed to work in **both Browser and Node.js environments**, with proper ESM and CommonJS support.

---

## Features

- Modern **ESM** and **CommonJS** support
- Works in **Browser** and **Node.js**
- Modular utility collection (use what you need)
- Safe environment checks (DOM APIs won’t crash Node)
- Canvas, DOM, Web API helpers included
- Zero dependencies
- Well-tested and predictable behavior
